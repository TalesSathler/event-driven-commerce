# Project Context

## Project Name

Event Driven Commerce

## Goal

Build a modern event-driven e-commerce platform for learning and portfolio purposes.

The project aims to demonstrate knowledge in:

* Java 21
* Spring Boot 3
* RabbitMQ
* PostgreSQL
* Angular
* Docker
* Microservices
* JWT Authentication
* Spec Driven Development

## Architecture

The system follows a microservice architecture.

Initial services:

* Product Service
* Inventory Service

Communication:

* REST APIs
* RabbitMQ Events

## Methodology

Spec Driven Development (SDD)

All work must follow:

Planning
→ Refinement
→ Task Generation
→ Implementation
→ Review

## Source of Truth

1. AGENTS.md
2. specs/
3. workflows/
4. docs/

## Current Phase

Phase 1

Focus:

* Authentication
* Product Management
* RabbitMQ Fundamentals
* Inventory Synchronization

## Non Goals

At this stage do not implement:

* Payments
* Notifications
* Kubernetes
* API Gateway
* Service Discovery

These features may be introduced in future phases.
